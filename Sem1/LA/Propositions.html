Formal Definition of Vector Space

Axioms

For u, v ∈ V and c, d ∈ F, the following axioms hold true:

  1. Closure under the operations: u+v ∈ V & cu ∈ V
  2. The following axioms hold for addition
    1. Associative property
    2. Identity
      • 0+u=u+0=u
    3. Every vector u ∈ V has an additive inverse v ∈ V such that u+v = 0
    4. Commutative property
  3. The following multiplication property are satisfied
    1. c(u+v) = cu+cv
    2. (c+d)u=cu+du
    3. c(du)=(cd)u
    4. 1u=u (where 1 is the identity unit element of F)

Example of Vector Space

  1. The space Rn of n-tuple of size n. These are often referred to as (column) vectors. The base field is R

  2. The space Rmxn of mxn matrix with real entries. Again with base field R
    Remark
    The fact that matrices form a vector space is of fundamental importance in both the theory and application of linear algebra
  3. The space C[0, 1] of continuous function from the closed interval [0, 1] on the real line of R, i.e.,
    C[0, 1]= {f: f is a continuous function, f:[0,1] -> R}
  4. The space R∞ of real sequences is a vector space over R, i.e.,

    R∞ = {<an>:<an> is a sequence with real numbers}

    Of more interest than R∞ itself is c, the subset of convergent sequences. It is also a vector space.

  5. The space Rn[t] of the polynomials of degree ≤ n with real coefficients.

  6. The space R[t] of all polynomial with all real coefficients.

  3. 0u = 0 for every vector u

  5. -u = (-1)u for every vector u

Subspace

Let V be a vector space over the field F. A (vector) subspace of V is a non-empty subset W of V, which is a vector space over F with the operations of vector additions and scalar multiplication taken from V.

Test for subspaces

Proposition 8

A subset W of V is a subset if and only if it satisfies the following three properties

  1. The zero vector 0 is in W
  2. W is closed under addition. That is, for each u & v ∈ W, the sum u+v ∈ W
  3. W is closed under scalar multiplication. That is, for each u ∈ W, and each scalar c, the scalar product cu ∈ W

Proposition 9

A non-empty subset W of V is a subspace if and only if for each u and v ∈ W, and each scalar c, the sum cu+v ∈ W.

Span of a set of vector

Definition 1

A linear combination of a finitely many given vectors in any sum of a scalar multiple of the vectors.

Definition 2

Let S = {v1, ..., vp} be a finite set of vectors in a vector space V. Then the span of S is the set of all the vectors that can be written as linear combinations of the vectors v1, ..., vp

Proposition 10

If S = {v1, ..., vp} is a set of vectors in a vector space V, then Span S = Span{v1, ..., vp} is a subspace of V.

Proof

Corollary 10.1

Let V be a vector space

  1. If U and W are two subspaces of V, then U∩W is also a subspace of V.
  2. If S = {v1, ..., vp} is a set of vectors in a vector space V, then Span S is the smallest subspace which contains S, i.e., if W is a subspace such that S⊆W, then Span S⊆W

Remark

In terms of this, Span S is sometimes described as the intersection of all subspaces of V containing S.

Linear Dependence

Let v1, v2, ..., vp be a finite list of vectors in a vector space V. Then the vectors are said to be (linearly) dependent if there exist scalars c1, c2, ..., cp, not all zeros, such that
c1v1+ ...+ cpvp = 0

Consequences of the Definition

  • Remark 1: Any list which contains the 0 has to be linearly dependent
  • Remark 2: A single non-zero vector is linearly independent
  • Remark 3: A list of two non-zero vectors is linearly dependent if and only if one vector is a scalar multiple of the other.
  • Remark 4: A list of non-zero vectors is linearly dependent if and only if at least one of the vectors is a linear combination of the others
  • Remark 5: Consequently, any list which contains a repeated vector must be linearly dependent. A list that is linearly independent corresponds to a set
  • Remark 6: Any list which contains a linearly dependent list is linearly dependent
  • Remark 7: Any subset of a linearly independent set is linearly independent.

Eigenvectors and Eigenvalues

An eigenvector of an nxn matrix A is a non-zero vector x such that Ax = λ\lambdaλx, for some scalar λ\lambdaλ.
The scalar λ\lambdaλ is called an eigenvalue of A if there is a non-trivial solution of Ax = λ\lambdaλx such a vector x is called an eigenvector corresponding to λ\lambdaλ

An eigenvector is not unique, since all scalar multiples of an eigenvector are also eigenvectors

Set of all eigenvectors corresponding to a fixed eigenvalue λ\lambdaλ of the nxn matrix A together with the zero vector forms a subspace of V=Fn
X= {v: Av=λ\lambdaλv}
=> X is a eigenspace of A corresponding to λ\lambdaλ

Two preliminary definition

Given an eigenvalues λ\lambdaλ1 for a matrix A we define:

  • The geometric multiplicity of λ\lambdaλ1 is the dimension of the eigenspace corresponding to λ\lambdaλ1

Let A be an nxn matrix with n(real) eigenvalues (counting algebraic multiplicity) of which λ\lambdaλ1, ..., λ\lambdaλp are distinct (p<n). Then the following holds true:

  2. A is diagonalizable, if and only if the geometric multiplicity for each λ\lambdaλk equal to its algebraic multiplicity
  3. If A is diagonalizable and Bk is a basis for the eigenspace corresponding to λ\lambdaλk for each k, then the total collection of vectors in B1, ..., Bp forms an eigenvectors basis for Rn

An inner product on a (real) vector space V is a function, that to each pair of vector u and v in V associates a scalar (real number) <u, v> and satisfies the following axioms for all vectors u, v, w in V and all scalars c in R

  1. <u, v> = <v, u>
  2. <u + v, w> = <u, w> + <v, w>
  3. <cu, v> = c<u, v>
  4. <u, u> ≥ 0, <u, u> = 0 if and only if u = 0

Two vectors u, v in V are said to be orthogonal to each other if <u, v> = 0; written as u⟂v

A set of vector {u1, ..., up} is said to be orthogonal if any two distinct vectors in the set are orthogonal to each other

Spectral theorem for symmetric matrices

Theorem 8

An nxn symmetric matrix A has the following properties:

  1. The eigenspace are mutually orthogonal
  2. A has n real eigenvalues, counting (algebraic) multiplicities
  3. A is orthogonally diagonalizable
  4. The dimension of the eigenspace for each eigenvalue of λ\lambdaλ equals the multiplicity of λ\lambdaλ, i.e., geometric multiplicity is equal to algebraic multiplicity

Corollary T8.1

A is orthogonally diagonalizable if and only if A is symmetric

Singular Value Decomposition

Observation 1

Let A be an mxn matrix. Then ATA, being a symmetric nxn matrix, can be orthogonally diagonalizable.

Let {v1, ..., vn} be an orthonormal basis for Rn consisting of eigenvectors of ATA with corresponding eigenvalues λ\lambdaλ1, ..., λ\lambdaλn. Then:
||Avi||2 = λ\lambdaλi

=> all eigenvalues of matrix ATA are non-negative
=> Eigenvalues can be rearranged so that: λ\lambdaλ1 ≥ ...≥ λ\lambdaλn ≥ 0

Proposition 51

Suppose that {v1, ..., vn} is an orthonormal basis for Rn consisting of eigenvectors of ATA with corresponding eigenvalues arranged so that λ\lambdaλ1 ≥ ...≥ λ\lambdaλn ≥ 0.
Suppose that A has r nonzero singular values then {Av1, ..., Avr} is an orthogonal basis for Col A and rank A = r

Proof

Definition

Let A be a mxn matrix.
The singular values of A are the square roots of the eigenvalues of ATA, denoted by σ1, σ2, ..., σn arranged in descending order, i.e., σi = √λ\lambdaλi for i = 1, ..., n

Note: The singular values are lengths of the vectors Av1, Av2, ..., Avn
